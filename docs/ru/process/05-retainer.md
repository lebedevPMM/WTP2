# Ретейнерная поддержка — текущие услуги

Подписная модель постоянного сопровождения клиента после завершения основной поставки. Применяется после завершения работ уровня L2 и выше.

---

## Уровни ретейнера

### Базовый ретейнер — $1,500/месяц

| Услуга | Периодичность |
|--------|---------------|
| Координация ежегодного продления визы | Ежегодно |
| Продление торговой лицензии | Ежегодно |
| Продление Emirates ID | По необходимости |
| Разовые консультации (email/звонок) | До 2 часов/месяц |
| Мониторинг CRM и отчёты о статусе | Ежемесячно |

**Подходит для:** Клиентов L1/L2 с простыми структурами, без сотрудников.

---

### Стандартный ретейнер — $3,000/месяц

| Услуга | Периодичность |
|--------|---------------|
| Всё из Базового | — |
| Бухгалтерия и учёт | Ежемесячно |
| Подача отчётности по НДС (VAT) | Ежеквартально |
| Подача отчётности по корпоративному налогу | Ежегодно |
| Продление виз сотрудников (до 3) | По необходимости |
| Ежеквартальный обзорный звонок с партнёром/клиентом | Ежеквартально |
| Управление взаимоотношениями с банком | Постоянно |

**Подходит для:** Клиентов L2 с активным бизнесом, 1–5 сотрудников.

---

### Премиум-ретейнер — $5,000/месяц

| Услуга | Периодичность |
|--------|---------------|
| Всё из Стандартного | — |
| Оформление виз сотрудников (до 10) | По необходимости |
| Выделенный аккаунт-менеджер | Постоянно |
| Консультации по управлению капиталом | Ежеквартально |
| Мониторинг портфеля недвижимости | Ежеквартально |
| Администрирование фонда DIFC | Ежегодно |
| Обновление завещания и планирование наследства | Ежегодно |
| Приоритетный ответ (< 4 часов) | Постоянно |
| Ежемесячный стратегический обзор | Ежемесячно |

**Подходит для:** Клиентов L3 со сложными структурами, несколькими юридическими лицами, потребностями семейного офиса.

---

## Что покрывает ретейнер и что оплачивается отдельно

| Включено в ретейнер | Оплачивается отдельно |
|---------------------|-----------------------|
| Координация и управление | Государственные сборы (транзитные) |
| Консультационное время (в рамках лимита) | Регистрация новых компаний |
| Подготовка документов | Дополнительные банковские счета |
| Обработка продлений | Сделки с недвижимостью |
| Отчётность о статусе | Судебные разбирательства / юридические споры |
| Взаимодействие с государственными органами | Поддержка при аудите и налоговых расследованиях |

---

## Условия ретейнерного соглашения

1. **Минимальный срок:** 6 месяцев
2. **Выставление счёта:** Ежемесячно, оплата до 1-го числа
3. **Неиспользованные часы:** Не переносятся
4. **Изменение объёма:** Требуют письменного согласования
5. **Расторжение:** Письменное уведомление за 30 дней после истечения минимального срока
6. **Государственные сборы:** Выставляются отдельно по фактической стоимости (без наценки)

## Доход партнёра от ретейнера

Партнёры получают рекуррентную долю от ретейнерного дохода по привлечённым клиентам:

| Уровень ретейнера | Ежемесячный доход | Доля партнёра (ориентировочно) |
|-------------------|-------------------|-------------------------------|
| Базовый | $1,500 | $150/месяц |
| Стандартный | $3,000 | $300/месяц |
| Премиум | $5,000 | $500/месяц |

**Годовой доход партнёра от одного клиента на Премиум-ретейнере:** $6,000/год рекуррентно.

## Переход от поставки к ретейнеру

| Шаг | Действие | Сроки |
|-----|----------|-------|
| 1 | Подтверждение завершения поставки | Неделя 0 |
| 2 | Отправка предложения по ретейнеру партнёру/клиенту | Неделя 1 |
| 3 | Выбор и подписание уровня ретейнера | Неделя 2 |
| 4 | Начало первого ежемесячного цикла | Неделя 3 |
| 5 | Назначение первого ежеквартального обзора | Месяц 3 |
