# Пакеты услуг — уровни L0–L3

WTP предлагает четыре уровня взаимодействия. Каждый уровень расширяет предыдущий. Рекомендуемый уровень определяется на этапе предварительной проверки на основании потребностей клиента и профиля риска.

---

## L0 — Консультация

**Для:** Клиентов, которым нужна ясность до перехода к исполнению.

| Параметр | Описание |
|----------|----------|
| **Объём** | Диагностика, оценка осуществимости, дорожная карта |
| **Результат** | Письменный консультационный отчёт с рекомендованным путём |
| **Продолжительность** | 3–5 рабочих дней |
| **Стоимость** | $1,500 – $3,000 |
| **Включает** | Предварительная проверка рисков, оценка банковского сценария, рекомендация по юрисдикции |
| **Не включает** | Любое исполнение, подачу документов или запись на приёмы |
| **Подходит для** | Клиенты с классификацией YELLOW, сложные структуры, клиенты, сравнивающие варианты |

**Когда рекомендовать L0:**
- У клиента нечёткие цели или несколько возможных путей
- Заключение по рискам — YELLOW: необходимо сначала установить условия
- Партнёр хочет задокументированную оценку осуществимости перед презентацией клиенту
- Клиент на раннем этапе планирования и не готов к обязательствам

---

## L1 — Старт

**Для:** Частных клиентов, которым нужна базовая регистрация в ОАЭ — виза, банковский счёт, налоговое резидентство.

| Параметр | Описание |
|----------|----------|
| **Объём** | Личное резидентство + банкинг + налоговая стратегия |
| **Результат** | Действующая виза, Emirates ID, личный банковский счёт, путь к налоговому резидентству |
| **Продолжительность** | 4–8 недель |
| **Стоимость** | $5,000 – $12,000 |

**Включённые услуги:**
- Оформление визы (инвестор/фрилансер/оценка Golden Visa)
- Оформление Emirates ID
- Открытие личного банковского счёта (1 банк, 1 тип счёта)
- Документ по стратегии налогового резидентства
- Банковский сценарий + координация встреч
- Маршрут поездки (если требуется визит)

**Не включено:**
- Регистрация компании
- Корпоративный банкинг
- Недвижимость
- Оформление членов семьи (доступно как дополнение)

---

## L2 — Регистрация

**Для:** Клиентов, которым нужна полная бизнес-регистрация с компанией, корпоративным банкингом и опционально недвижимостью.

| Параметр | Описание |
|----------|----------|
| **Объём** | Компания + корпоративный счёт + личная регистрация + опционально недвижимость |
| **Результат** | Зарегистрированная компания, торговая лицензия, корпоративный банковский счёт, виза, Emirates ID |
| **Продолжительность** | 6–12 недель |
| **Стоимость** | $15,000 – $35,000 |

**Включённые услуги:**
- Всё из L1
- Регистрация компании (Mainland или Free Zone)
- Торговая лицензия
- Открытие корпоративного банковского счёта (1 банк)
- Регистрация корпоративного налога
- Зарегистрированный агент / PRO-услуги (первый год)
- Консультации по недвижимости (поиск + координация просмотров)
- Оформление семейных виз (супруг(а) + дети)
- Поддержка в поступлении в школу (исследование + подача заявок)

**Доступные дополнения:**
- Дополнительные банковские счета (+$2,000–5,000 за банк)
- Сопровождение сделки с недвижимостью (+$3,000–8,000)
- Координация ипотеки (+$2,000)

---

## L3 — Управление

**Для:** Клиентов с постоянными операционными потребностями — бухгалтерия, управление персоналом, структурирование капитала.

| Параметр | Описание |
|----------|----------|
| **Объём** | Полное операционное сопровождение + текущая поддержка + управление капиталом |
| **Результат** | Полностью функционирующая компания с бухгалтерией, HR и структурами управления капиталом |
| **Продолжительность** | Первоначальная настройка 8–16 недель, затем текущее сопровождение |
| **Стоимость** | $35,000 – $80,000+ (первоначально) + ретейнер |

**Включённые услуги:**
- Всё из L2
- Настройка бухгалтерии и учёта
- Регистрация и подача отчётности по НДС (VAT)
- Оформление виз сотрудников (до 5)
- Консультации по управлению капиталом
- Создание фонда в DIFC (защита активов)
- Координация кастодиального счёта
- Регистрация завещания (суды DIFC)
- Ежеквартальные обзорные встречи
- Выделенный аккаунт-менеджер

**Ретейнер (после настройки):**
- Ежемесячный ретейнер: $2,000 – $5,000/месяц
- Покрывает: ежегодные продления, текущий учёт, поддержка HR, разовые консультации

## Гид по выбору пакета

| Потребность клиента | Рекомендуемый уровень |
|---------------------|----------------------|
| «Я просто хочу понять свои возможности» | L0 — Консультация |
| «Мне нужна виза и банковский счёт» | L1 — Старт |
| «Я переношу свой бизнес в ОАЭ» | L2 — Регистрация |
| «Мне нужна виза для конкретной цели» | L1 — Старт |
| «Я хочу купить недвижимость и зарегистрировать компанию» | L2 — Регистрация |
| «Мне нужно полное сопровождение: компания, бухгалтерия, сотрудники» | L3 — Управление |
| «Я хочу защиту активов и структурирование капитала» | L3 — Управление |
| «У меня сложный профиль / риск YELLOW» | Сначала L0, затем переоценка |

---

## Путь апсейла

```
L0 Консультация → L1 Старт → L2 Регистрация → L3 Управление
       ↓               ↓             ↓                ↓
   Дорожная карта  Виза+Банк   Компания+Недвижимость  Операции
```

Завершение каждого уровня создаёт естественный переход к обсуждению следующего:
- L0 → «На основе дорожной карты вот что покроет L1»
- L1 → «Теперь, когда банкинг налажен, регистрация компании — логичный следующий шаг»
- L2 → «Ваша компания работает — давайте настроим бухгалтерию и визы для сотрудников»
