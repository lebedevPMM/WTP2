# Пакеты услуг — Уровни 0–3

WTP предлагает четыре уровня сотрудничества. Каждый последующий уровень расширяет предыдущий. Рекомендуемый уровень определяется на этапе предварительной проверки с учетом потребностей клиента и его рискового профиля.

---

## Level 0 — Advisory

**Для кого:** Клиенты, которым нужна ясность перед принятием решений.

| Параметр | Описание |
|----------|----------|
| **Объем** | Диагностика, оценка целесообразности, дорожная карта |
| **Результат** | Письменный консультационный отчет с рекомендуемым планом действий |
| **Срок** | 3–5 рабочих дней |
| **Стоимость** | $1,500 – $3,000 |
| **Включает** | Предварительная оценка рисков, анализ банковских сценариев, рекомендации по юрисдикции |
| **Не включает** | Любое исполнение, подачу документов или назначение встреч |
| **Подходит для** | Клиентов с классификацией YELLOW, сложных структур, клиентов, сравнивающих варианты |

**Когда рекомендовать L0:**
- У клиента нечеткие цели или несколько возможных путей
- Рисковый вердикт — YELLOW: сначала необходимо установить условия
- Партнер хочет получить задокументированную оценку целесообразности перед представлением клиенту
- Клиент находится на ранней стадии планирования и не готов к обязательствам

## Level 1 — Entry

**Для кого:** Частные клиенты, которым нужна базовая настройка в ОАЭ — виза, банковский счет, налоговое резидентство.

| Параметр | Описание |
|----------|----------|
| **Объем** | Личное резидентство + банкинг + налоговая стратегия |
| **Результат** | Действующая виза, Emirates ID, личный банковский счет, путь к налоговому резидентству |
| **Срок** | 4–8 недель |
| **Стоимость** | $5,000 – $12,000 |

**Включенные услуги:**
- Подача заявления на визу (инвесторская/фрилансерская/оценка Golden Visa)
- Оформление Emirates ID
- Открытие личного банковского счета (1 банк, 1 тип счета)
- Документ по стратегии налогового резидентства
- Банковский сценарий + координация встречи
- Маршрут поездки (если требуется визит)

**Не включено:**
- Регистрация компании
- Корпоративный банкинг
- Недвижимость
- Оформление членов семьи (доступно как дополнительная опция)

---

## Level 2 — Setup

**Для кого:** Клиенты, которым нужна полная настройка бизнеса: компания, корпоративный банкинг и опционально недвижимость.

| Параметр | Описание |
|----------|----------|
| **Объем** | Компания + корпоративный счет + личная настройка + опционально недвижимость |
| **Результат** | Зарегистрированная компания, торговая лицензия, корпоративный банковский счет, виза, Emirates ID |
| **Срок** | 6–12 недель |
| **Стоимость** | $15,000 – $35,000 |

**Включенные услуги:**
- Все услуги Level 1
- Регистрация компании (Mainland или Free Zone)
- Торговая лицензия
- Открытие корпоративного банковского счета (1 банк)
- Регистрация корпоративного налога
- Зарегистрированный агент / PRO-услуги (первый год)
- Консультации по недвижимости (поиск + координация просмотров)
- Оформление виз для семьи (супруг(а) + дети)
- Поддержка при поступлении в школы (исследование + подача заявлений)

**Дополнительные опции:**
- Дополнительные банковские счета (+$2,000–5,000 за банк)
- Юридическое сопровождение покупки недвижимости (+$3,000–8,000)
- Координация ипотечного кредитования (+$2,000)

---

## Level 3 — Control

**Для кого:** Клиенты с постоянными операционными потребностями — бухгалтерия, управление сотрудниками, структурирование активов.

| Параметр | Описание |
|----------|----------|
| **Объем** | Полное операционное сопровождение + постоянная поддержка + управление капиталом |
| **Результат** | Полностью функционирующая организация с бухгалтерией, HR и структурами для управления активами |
| **Срок** | Первоначальная настройка 8–16 недель, далее — постоянное сопровождение |
| **Стоимость** | $35,000 – $80,000+ (первоначальная) + ретейнер |

**Включенные услуги:**
- Все услуги Level 2
- Настройка бухгалтерского учета
- Регистрация и подача отчетности по VAT
- Оформление виз для сотрудников (до 5 человек)
- Консультации по управлению капиталом
- Создание DIFC Foundation (защита активов)
- Координация открытия кастодиального счета
- Регистрация завещания (DIFC Courts)
- Ежеквартальные обзорные встречи
- Персональный менеджер

**Ретейнер (после настройки):**
- Ежемесячный ретейнер: $2,000 – $5,000/мес.
- Покрывает: ежегодные продления, текущий бухучет, HR-поддержку, ad-hoc консультации

## Руководство по выбору пакета

| Потребность клиента | Рекомендуемый уровень |
|---------------------|----------------------|
| «Я просто хочу понять свои возможности» | L0 — Advisory |
| «Мне нужна виза и банковский счет» | L1 — Entry |
| «Я переношу свой бизнес в ОАЭ» | L2 — Setup |
| «Мне нужна виза для конкретной цели» | L1 — Entry |
| «Я хочу купить недвижимость и открыть компанию» | L2 — Setup |
| «Мне нужны полные операции: компания, бухгалтерия, сотрудники» | L3 — Control |
| «Я хочу защиту активов и структурирование капитала» | L3 — Control |
| «У меня сложный профиль / YELLOW-риск» | Сначала L0, затем пересмотр |

## Путь апсейла

| Уровень | Что включает | Путь к следующему |
|---------|-------------|-------------------|
| L0 Advisory | Дорожная карта | «На основе дорожной карты — вот что покроет L1» |
| L1 Entry | Виза + Банк | «Банкинг настроен — регистрация компании логичный следующий шаг» |
| L2 Setup | Компания + Недвижимость | «Компания работает — давайте настроим бухгалтерию и HR» |
| L3 Control | Операционное управление | Полная операционная поддержка |

Завершение каждого уровня создает естественную возможность для перехода на следующий.
