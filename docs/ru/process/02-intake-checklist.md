# Чек-лист приема — KYC Light

Структурированная анкета предварительной проверки. Заполняется партнером или клиентом до начала работы. Это основной источник данных для классификации рисков и выбора банковского сценария.

---

## Раздел 1: Цель обращения

- Какова основная цель вашего обращения по ОАЭ?
  - [ ] Регистрация компании + корпоративный банкинг
  - [ ] Личный банкинг + резидентство
  - [ ] Полная релокация (семья + бизнес)
  - [ ] Защита активов / структурирование капитала
  - [ ] Оптимизация налогового резидентства
  - [ ] Другое: ___________

- Каковы ожидаемые сроки?
  - [ ] Срочно (< 1 месяца)
  - [ ] Стандартно (1–3 месяца)
  - [ ] Этап планирования (3–6 месяцев)

## Инструкция для партнеров

1. Заполните все **обязательные** поля — необязательные поля помогают уточнить сценарий
2. Приложите: копию паспорта (главная страница), документацию SOF (банковские выписки или письмо бухгалтера)
3. Отправьте через вашего менеджера WTP или по согласованному каналу связи
4. Ожидаемые сроки: заключение по рискам в течение **5–7 рабочих дней после получения полного комплекта запрашиваемых документов**
5. Неполные заявки будут возвращены с запросом на недостающие документы
