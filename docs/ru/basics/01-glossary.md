# Глоссарий и терминология

Единый справочник терминов, используемых во всех документах WTP, на лендинге, в коммуникациях с партнёрами и внутренних процессах.

---

## Основные бизнес-термины

| Термин | Определение |
|--------|------------|
| **WTP** | Операционный партнёр в ОАЭ, специализирующийся на регистрации компаний, корпоративном банкинге и комплаенсе для международных клиентов |
| **Execution Partner (операционный партнёр)** | Локальный оператор, который ведёт полный цикл реализации — от намерения до результата — от имени брокеров и консультантов |
| **Partner / Broker (партнёр / брокер)** | Внешний консультант, управляющий активами, семейный офис или иммиграционный консультант, направляющий клиентов в WTP |
| **Client (клиент)** | Конечный бенефициар — физическое или юридическое лицо, для которого осуществляется релокация/структурирование в ОАЭ |
| **Referral Model (реферальная модель)** | Партнёр представляет клиента → WTP выполняет работу → партнёр получает комиссию |
| **White-Label Model (модель white-label)** | WTP работает под брендом партнёра — клиент не взаимодействует с WTP напрямую |

---

## Процесс и реализация

| Термин | Определение |
|--------|------------|
| **Banking-First Model (модель «сначала банк»)** | Ключевой подход WTP: оценка банковской пригодности до начала любого корпоративного оформления |
| **Pre-Screen (предварительная проверка)** | Первичная проверка благонадёжности: KYC, AML, источник средств, классификация рисков |
| **Banking Scenario (банковский сценарий)** | Рекомендованный банк + тип счёта + требования на основе профиля клиента |
| **Intake Checklist (входная анкета)** | Структурированный опросник KYC Light, заполняемый до начала работы |
| **Risk Verdict (заключение о рисках)** | Результат классификации: GREEN (одобрен), YELLOW (условно), RED (отказ) |
| **Process Map (карта процесса)** | 10-этапный рабочий процесс от входа до постоянного сопровождения |
| **Delivery Package (пакет услуг)** | Комплексный набор услуг (Level 0–3), адаптированный под потребности клиента |
| **Retainer (абонентское обслуживание)** | Постоянная подписочная поддержка после первоначальной реализации |

## Риски и комплаенс

| Термин | Определение |
|--------|------------|
| **KYC** | Know Your Customer — верификация личности и проверка благонадёжности |
| **AML** | Anti-Money Laundering — процедуры противодействия отмыванию денег |
| **SOF / SOW** | Source of Funds / Source of Wealth — документальное подтверждение происхождения средств клиента |
| **Red Flag (красный флаг)** | Предупреждающий индикатор в профиле клиента, требующий углублённой проверки |
| **Red Flags Matrix (матрица красных флагов)** | Систематический чек-лист из 8 категорий рисков, применяемый при предварительной проверке |
| **Sanctions Screening (санкционный скрининг)** | Проверка по международным санкционным спискам (ООН, ЕС, OFAC, ОАЭ) |
| **PEP** | Politically Exposed Person — политически значимое лицо, требующее углублённой проверки |
| **UBO** | Ultimate Beneficial Owner — конечный бенефициарный владелец, физическое лицо, контролирующее юридическое лицо |
| **Substance (экономическое присутствие)** | Доказательство реальной деятельности компании в ОАЭ (офис, сотрудники, транзакции) |
| **Nominee Structure (номинальная структура)** | Использование третьих лиц для сокрытия истинного владения — красный флаг |

## Продукты и услуги

| Термин | Определение |
|--------|------------|
| **Company Formation (регистрация компании)** | Регистрация юридического лица в ОАЭ (Mainland, Free Zone или Offshore) |
| **Free Zone (свободная зона)** | Особая экономическая зона со 100% иностранным владением и налоговыми льготами |
| **Mainland (материковая юрисдикция)** | Стандартная регистрация компании в ОАЭ через DED (Department of Economic Development) |
| **Corporate Account (корпоративный счёт)** | Банковский счёт для юридического лица |
| **Personal Account (личный счёт)** | Банковский счёт для физического лица |
| **Premium Banking (премиальный банкинг)** | Банковский сервис для состоятельных клиентов (Citi Gold, HSBC Premier, Emirates NBD Private) |
| **Golden Visa (золотая виза)** | 10-летняя резидентская виза ОАЭ для инвесторов, предпринимателей и специалистов |
| **Emirates ID** | Национальное удостоверение личности ОАЭ — обязательно для всех резидентов |
| **Conveyancing (конвейансинг)** | Юридический процесс передачи права собственности на недвижимость |
| **Foundation (фонд)** | Структура для защиты активов (DIFC Foundation) в целях наследственного планирования |
| **Custody Account (кастодиальный счёт)** | Инвестиционный счёт для хранения ценных бумаг и активов |

## Уровни пакетов

| Уровень | Название | Объём |
|---------|----------|-------|
| **L0** | Advisory | Диагностика, дорожная карта, оценка осуществимости |
| **L1** | Entry | Виза + личный банковский счёт + налоговая стратегия |
| **L2** | Setup | Компания + корпоративный счёт + недвижимость + релокация семьи |
| **L3** | Control | Бухгалтерия + визы для сотрудников + управление капиталом + кастодиальный счёт |

## Партнёрские термины

| Термин | Определение |
|--------|------------|
| **Partner Kit (партнёрский комплект)** | Брендированный набор материалов для онбординга партнёра (одностраничник, процесс, FAQ) |
| **Pilot Program (пилотная программа)** | Пробный период партнёрства — как правило, 2–3 кейса — до полного подтверждения сотрудничества |
| **SLA** | Service Level Agreement — соглашение о сроках реагирования и обязательствах по реализации |
| **Commission (комиссия)** | Вознаграждение партнёра за направленного клиента, выплачиваемое после достижения этапа реализации |
| **Client Protection (защита клиента)** | Гарантия, что WTP не будет работать с клиентом в обход направившего партнёра |
| **CRM Tagging (тегирование в CRM)** | Все клиенты, направленные партнёром, помечаются в Bitrix24 CRM идентификатором партнёра |
