# Шаблон письма — Пилотная программа

Для приглашения заинтересованных партнёров начать с пилотного кейса до оформления полноценного партнёрства.

---

## Версия A: Предложение пилота после позитивного звонка

**Тема:** Пилотная программа — начнём с одного кейса

Здравствуйте, [Имя],

По итогам нашего разговора думаю, что оптимальный вариант — пилот: один кейс, без долгосрочных обязательств, чтобы вы могли оценить нашу работу на практике.

**Как устроен пилот:**

1. **Вы отправляете профиль клиента** — через нашу анкету или по email
2. **Мы проводим бесплатную предварительную оценку** — KYC, классификация рисков, банковский сценарий (5–7 рабочих дней)
3. **Вы изучаете результат** — если всё подходит, переходим к исполнению
4. **Мы реализуем** — в рамках согласованного объёма и сроков
5. **Подводим итоги** — что сработало, что скорректировать

**Чем пилот отличается:**
- Бесплатная предварительная оценка (без затрат на оценку рисков)
- Прямой доступ к аккаунт-менеджеру (ко мне)
- Полная прозрачность по процессу и ценообразованию
- Никаких обязательств по партнёрству до тех пор, пока вы не будете удовлетворены

**Что нам потребуется от вас:**
- Один клиентский профиль в рамках нашей специализации (регистрация компании, банкинг или резидентство в ОАЭ)
- Заполненный чек-лист (я отправлю форму)
- Копия паспорта и документация по SOF для клиента

Готовы попробовать? Просто ответьте на это письмо, и я вышлю анкету.

С уважением,
[Ваше имя]

## Версия B: Предложение пилота новому лиду

**Тема:** Бесплатная предварительная оценка — протестируйте наш сервис в ОАЭ

Здравствуйте, [Имя],

Если у вас есть клиенты, рассматривающие регистрацию компании, банкинг или резидентство в ОАЭ, я хотел бы предложить бесплатную предварительную оценку — как способ познакомиться с нашим сервисом.

**Без обязательств.** Вот что вы получите:

- Классификация риска клиентского профиля (GREEN / YELLOW / RED)
- Рекомендация по банковскому сценарию (какой банк, какие требования)
- Рекомендация по пакету услуг с ориентировочными сроками и стоимостью
- Всё — в течение 5–7 рабочих дней

Если результат оценки положительный и клиент готов двигаться дальше, мы берём на себя полный цикл исполнения. Вы получаете комиссию по факту реализации — без операционной нагрузки.

Если не подойдёт — без затрат и без обязательств.

Интересно? Отправьте мне профиль клиента, и я приступлю.

С уважением,
[Ваше имя]

## Версия C: Оформление партнёрства после успешного пилота

**Тема:** Партнёрское соглашение — следующий шаг

Здравствуйте, [Имя],

Пилотный кейс прошёл успешно — [краткое описание результата]. Благодарю за продуктивное сотрудничество.

Хотел бы оформить наше партнёрство. Вот что включает соглашение:

- **Структура комиссии:** [X%] от стоимости услуг WTP по направленным клиентам
- **Защита клиента:** Все направленные клиенты тегируются в нашей CRM, прямое взаимодействие — только с вашего разрешения
- **Отчётность:** Еженедельные обновления по активным кейсам, ежемесячная сводка
- **Формат:** Реферальная модель / White-label (как обсуждали)
- **Срок:** 12 месяцев, автоматическое продление, 30-дневное уведомление о расторжении

Подготовлю проект соглашения на этой неделе. В промежутке — можете направлять новые кейсы: пилотные условия действуют до подписания соглашения.

С уважением,

## Детали пилотной программы (внутренний справочник)

| Аспект | Условия пилота | Условия полного партнёрства |
|--------|---------------|---------------------------|
| Стоимость предварительной оценки | Бесплатно | Бесплатно |
| Ставка комиссии | Стандартная | Стандартная или по договорённости |
| Количество кейсов | 1–3 | Без ограничений |
| Отчётность | По каждому кейсу | Еженедельная + ежемесячная |
| Необходимость соглашения | Нет | Да |
| Срок действия | До совместного решения | 12 месяцев с продлением |
| Уровень поддержки | Личный аккаунт-менеджер | Выделенный или общий |
